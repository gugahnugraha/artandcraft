# Features Directory

This directory contains self-contained modules representing specific business features of ArtAndCraft.id:
- `auth`: Credentials & OAuth login, registration, roles (Buyer, Seller, Admin).
- `products`: Catalog browsing, product cards, categories, full-text search.
- `store`: Seller store landing, banners, profile management.
- `cart`: Shopping cart items, local storage sync, checkout.
- `orders`: Invoicing, status tracking, payment callback handling.
- `admin`: Platform settings, user moderation, reports, coupons.
